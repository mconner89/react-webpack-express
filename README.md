# react-webpack-express
This is a super basic setup to get started with react, webpack, babel, express.
#### To Get Started
1. ` git clone https://github.com/ptbarnum4/react-webpack-express.git`
2. `cd react-webpack-express`
3. `npm install`
4. `npm run dev`
5. `npm run start`
***
